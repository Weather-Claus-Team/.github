## 프로젝트 소개
웨더클로스 프로젝트는 날씨에 따른 옷차림을 추천해주는 웹사이트입니다. <br>
검색이나 위치 동의를 통해 원하는 지역의 날씨를 확인하고 옷차림 추천을 받아보세요 <p>
